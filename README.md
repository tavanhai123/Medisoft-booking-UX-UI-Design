# Thiết kế trải nghiệm đặt lịch khám online – Medisoft (UI/UX Project)

## Tóm tắt dự án
Dự án thiết kế trải nghiệm và giao diện cho hệ thống đặt lịch khám trực tuyến dành cho phòng khám tư nhân. Mục tiêu: tối ưu hành trình người dùng (booking flow), giảm friction, minh bạch thông tin dịch vụ và nâng cao hiệu quả vận hành cho phòng khám.

## Những sản phẩm chính (BA / UX deliverables)
- Báo cáo nghiên cứu người dùng & kết quả khảo sát (insights, pain points).  
- User Persona, Empathy Map, Customer Journey Map.  
- Problem statement, POV, How Might We.  
- User Flow, Sitemap, Storyboard.  
- Wireframes (mid-fi → high-fi) và style guide.  
- Prototype tương tác (Figma link).  
- Test cases giao diện và kịch bản kiểm thử usability.  
- Tổng kết & đề xuất cải tiến (KPI, out-of-scope, rủi ro, giả định).
